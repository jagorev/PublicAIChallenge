# AI Catasto Challenge

## Contesto

Questo progetto nasce da una challenge ([Public AI Challenge - HIT](https://www.trentinoinnovation.eu/innova/strumenti-per-innovazione/public-ai-challenge/)) incentrata sulla creazione di un sistema di Machine Learning a supporto degli operatori del catasto nella classificazione delle unità immobiliari.

## Obiettivi

Il progetto ha i seguenti obbiettivi:

1.  **Predizione automatizzata**: Sviluppare un modello di Machine Learning in grado di predire la classe e categoria catastale di nuove unità immobiliari a partire da descrizioni delle unità.

2. **Integrazione con QGIS**: Creare un plug-in Python per QGIS3 che permetta agli operatori del catasto di visualizzare e utilizzare le funzionalità del sistema direttamente nell'ambiente di lavoro già in uso.
